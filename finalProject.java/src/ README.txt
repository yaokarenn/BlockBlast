My project is inspired by the popular mobile game "block blast." It is a puzzle game, in which
you drag shapes and drop the into a grid. When a row or a column is filled, then the filled in blocks
clears and you earn points. However, if there are no more places in the grid to put your shape, you
lose. The purpose of this game is to get the most amount of points. I included a score calculator and
three different shapes you can choose from. I am most proud of the grid class.
